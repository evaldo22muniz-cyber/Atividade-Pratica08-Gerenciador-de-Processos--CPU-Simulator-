# Atividade-Pratica08-Gerenciador-de-Processos--CPU-Simulator-
Cenário: O processador do computador precisa decidir qual tarefa executar. Tarefas de "Sistema" (Alta prioridade) passam na frente de tarefas de "Usuário" (Baixa prioridade). Você usará uma lista que se auto-ordena (simulando uma Heap/Priority Queue) para gerenciar isso.
